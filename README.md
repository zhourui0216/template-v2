# vue2-项目模板

## 项目介绍
模板

### 注意事项
- 如需要移动端布局，请在项目文件夹下postcss.config.js中解开配置，并安装依赖

### 环境依赖
- git
- nodejs

### 项目运行
- 下载依赖: npm install
- 运行: npm run serve
- 打包: npm run build

### 技术选型
 技术 | 说明
 ---- | ----
 Vue2 | Vue框架
 VueX | 状态管理
 axios | 请求库

### 项目结构
```
└── src

    ├── api -- 封装的接口

    ├── assets -- 静态资源

    ├── components -- 封装的组件

    ├── router -- 路由

    ├── store -- 状态管理

    ├── utils -- 方法类

    ├── view -- 视图文件

    ├──  App

    └──  main
```
